# Veris – AI Legal Assistant

**Veris** is a full-stack AI-powered web app that helps users understand legal documents, generate clauses, and get plain-language explanations of legal concepts.

---

## 🛠 Tech Stack

### Backend:
- FastAPI (Python)
- Uvicorn
- REST API

### Frontend:
- ReactJS
- Vite
- JavaScript

---

## 📁 Project Structure

```
veris/
├── backend/       → FastAPI backend (Python)
├── frontend/      → ReactJS frontend (Vite)
├── README.md
└── .gitignore
```

---

## 🚀 How to Run the App

### Backend (FastAPI):
```bash
cd backend
source venv/bin/activate
uvicorn app.main:app --reload
```

### Frontend (React):
```bash
cd frontend
npm install
npm run dev
```
